#### list数据类型简单使用
```bash
$ lpush k1 a b c d             # 左推进(从左边开始加数据)
$ rpush k1 a b c d             # 右推进(从右边开始加数据)
$ lrange k1 0 -1               # 左查看，0 -1是范围，就是从第0个下标开始到-1，就是全部的数据(从左边开始查看数据)
$ lpop k1                      # 左推出(从左边开始取出数据，取出来这个数据就没有了)
$ rpop k1                      # 右推出(从右边开始取出数据，取出来这个数据就没有了)
$ rpoplpush k1 g               # 右推出1个，再左推进1个g
$ lindex k1 1                  # 从左边开始取出下标为1的数据
$ lset k1 1 c                  # 从左边开始将下标为1的数据替换为c
$ llen k1                      # list的长度
$ lrem k1 2 ab                 # 删除2个值为ab的数据(0删除值为ab的全部数据，-1从右边开始删除值为ab的数据)
$ ltrim k1 0 1                 # 删除下标0到1以外的所有数据
$ linsert k1 before t a        # 在t的前面插入数据a
$ linsert k1 after t a         # 在t的后面插入数据a
$ blpop k1 100                 # 阻塞等待 lpush 命令加如的数据(100是等待100秒，如果是0就永久等待，除非有数据加入)  
```
